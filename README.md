# 🚀 Primeros Pasos con Flutter - Namer App

¡Bienvenido a mi primer proyecto en Flutter! Esta aplicación es el resultado de seguir la documentación oficial para aprender los conceptos fundamentales del desarrollo móvil multiplataforma.

## 📝 Descripción

**Namer App** es una aplicación sencilla pero potente que genera combinaciones aleatorias de palabras en inglés. Permite a los usuarios marcar sus favoritas y visualizarlas en una lista dedicada. 

Este proyecto cubre:
- Arquitectura básica de Flutter.
- Gestión de estado con `provider`.
- Uso de paquetes externos (`english_words`).
- Diseño de interfaces responsivas y modernas.

## ✨ Características

- **Generación Aleatoria:** Crea nombres únicos con un solo clic.
- **Favoritos:** Guarda los nombres que más te gusten.
- **Navegación:** Cambia entre el generador y tu lista de favoritos.
- **Interfaz Adaptable:** Funciona en Android, iOS, Web y Escritorio.

## 🛠️ Requisitos Previos

Antes de comenzar, asegúrate de tener instalado:
- [Flutter SDK](https://docs.flutter.dev/get-started/install) (Versión 3.0.0 o superior)
- [Dart SDK](https://dart.dev/get-started/sdk)
- Un editor de código como **VS Code** o **Android Studio**.

## 🚀 Instalación y Uso

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/LeonardoDiazO/FirstAppflutterDocumentation.git
   ```

2. **Entrar al directorio:**
   ```bash
   cd FirstAppflutterDocumentation
   ```

3. **Instalar dependencias:**
   ```bash
   flutter pub get
   ```

4. **Ejecutar la aplicación:**
   ```bash
   flutter run
   ```

## 📱 Visualización

Puedes desplegar esta aplicación en:
- Emuladores de **Android/iOS**.
- Navegadores Chrome/Edge (soporte Web).
- Aplicaciones nativas de escritorio (Windows, macOS, Linux).
